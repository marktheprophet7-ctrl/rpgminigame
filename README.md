# Mini RPG (Vanilla JS)

A small browser RPG with:
- tile-based exploration
- random encounters
- turn-based combat
- leveling + XP
- loot chests + potions
- save/load via LocalStorage

## Run locally
Simplest: open `index.html` in your browser.

Recommended (VS Code):
1. Install the extension **Live Server**
2. Right click `index.html` â†’ **Open with Live Server**

## Controls
- Move: WASD / Arrow keys
- Interact: E
- Inventory info: I
- Toggle log: L
- Combat: 1 Attack, 2 Defend, 3 Heal, 4 Run

## Save/Load
Uses `localStorage` under the key `mini_rpg_save_v1`.
